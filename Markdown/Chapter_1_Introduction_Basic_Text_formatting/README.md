## Chapter 1: Introduction & Basic Text Formatting

### The Big Concept: What You Type vs. What You See

Unlike traditional word processors (like Microsoft Word or Google Docs) where you click buttons to make text bold or change font sizes, Markdown uses plain text symbols.

A "Markdown parser" reads these symbols and instantly translates them into clean, universally accessible HTML. This means your files stay lightweight, future-proof, and easily readable even as raw text.

---

### Step 1: Headings

Headings organize the hierarchy of your document. In Markdown, you create headings using the hash or pound symbol (`#`).

The number of hashes you use corresponds to the heading level (from 1 to 6). **Crucial Rule:** You must always put a space between the hashes and your heading text!

#### How to write it:

```markdown
# Heading 1 (The main title)
## Heading 2 (Major sections)
### Heading 3 (Sub-sections)
#### Heading 4
##### Heading 5
###### Heading 6 (The smallest sub-heading)

```

---

### Step 2: Emphasis (Italics & Bold)

To grab a reader's attention, you can make text italicized or bold using asterisks (`*`) or underscores (`_`).

#### 1. Italics

Wrap your text in a *single* asterisk or underscore.

```markdown
This is *italicized* text using asterisks.
This is also _italicized_ text using underscores.
```

#### 2. Bold

Wrap your text in *double* asterisks or underscores.

```markdown
This is **bold** text using asterisks.
This is also __bold__ text using underscores.
```

#### 3. Bold & Italic

Want to use both at once? Wrap your text in *triple* asterisks.

```markdown
This is ***incredibly important*** text!
```

---

### Step 3: Strikethrough

If you want to show that something has been deleted, crossed off, or corrected, you can apply a strikethrough by wrapping your text in two tildes (`~~`).

#### How to write it:

```markdown
The world is ~~flat~~ a sphere.
```

### Putting It All Together

If a computer were to render the syntax we just learned, it would look like this:

> # Heading 1 (The main title)
> 
> 
> ## Heading 2 (Major sections)
> 
> 
> This is a normal paragraph showing how you can easily combine **bold actions**, *italic notes*, and ~~mistakes~~ corrections seamlessly.
